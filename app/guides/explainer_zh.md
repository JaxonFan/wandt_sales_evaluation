# 你的奖金是怎么算的（大白话版）

你的收入 = **底薪** + **奖金**。奖金奖励三件事——每一项你都可以自己核对。

## 1. 卖货（订单行数）
你写在订单上的**每一行**都能拿到一点点钱。卖得越多、每张订单卖的**品类**越多，这部分就越高。
金额特意设得小——底薪已经覆盖了你上班和维护客户的工作，这只是额外的一点辛苦费。

## 2. 把客户做大（重头戏）
我们给你一个**目标**，你**超过它**就能拿奖金。目标是这样定的，说白了就是：

> **你的目标 = 你的客户去年同样这 3 个月买了多少，再按和你规模相近的客户今年实际增长了多少往上调，
> 最后加一点点冲刺幅度。**

举个例子：

- 你的客户去年这 3 个月买了 **$520,000**。
- 和你规模相近的客户今年大约涨了 **+10%**（市场在涨），所以门槛抬到约 $572k。
- 再加一点 **+3% 冲刺** → 你的目标大约是 **$590,000**。
- 如果你最近 3 个月做到 **$620,000**，就比门槛高出 $30,000 → 这就是你的增长奖金。

**为什么要看"和你规模相近的客户涨了多少"？** 如果整个市场都在涨，你只是顺势而为，那并不是**你**跑赢了——
所以你的目标已经跟着市场往上调了。增长奖金奖励的是你**跑赢同类客户**，而不是奖励大盘上涨。
（反过来，如果市场在跌，你的目标也会跟着降，所以大家一起遇到的低谷不会怪到你头上。）

**为什么看 3 个月而不是 1 周？** 销售一周一周地波动很大——一笔大单或者一个淡周不应该让你的奖金大起大落。
我们看滚动的 3 个月，这样更稳。

**兼职怎么办？** 你的冲刺幅度会按你实际工作的小时数缩放，所以不会拿全职的标准来要求你。

**"我达成目标了，为什么没有增长奖金？"** 达成目标说明你**踩在节奏上**，这部分已经付过了——
靠底薪、按行计酬、以及新客户的奖励。增长奖金是**超出目标之后的额外奖励**。
就像配额：完成是本职，超额才有奖金。

**"我比上个季度卖得多，奖金反而少了，为什么？"** 多半是门槛抬高了（市场更强），所以要超过它得卖更多。
你是和门槛比，不是和去年的一个固定数字比。

**某个大客户突然暴涨：** 如果单个客户猛涨（比如一次性大批量采购，或者刚划给你），
我们会计到一个合理上限，**剩下的部分标记出来交给经理复核**——
这样一个走运的客户不会决定你的成绩，真正的好成绩经过复核后也照样被认可。

## 3. 新客户
- **你拿下的（自己开发）：** 这个客户**头 3 个月左右买的所有东西**你都能分到一份——
  客户越大，分得越多。之后它就并入你的常规客户群。
- **派给你的（分配）：** 不是你开发的，所以没有新客户分成——但你现在做它依然能拿订单行的钱，
  等它积累几个月历史后，就开始按**增长**计（和它自己最近的表现比，因为还没有去年同期数据）。

## 名词解释
- **自己开发**（新）——你拿下的；头 1 个季度左右享受新客户分成。
- **分配**（新）——派给你的；现在拿订单行的钱，有了历史后按增长计（没有新客户分成）。
- **临时**——历史不足一年；增长和它**自己上一个季度**比。
- **成熟**——满一年及以上；增长和**去年同一季度**比，并按规模同类客户做调整。
- **封顶**——增长超过目标的 2 倍；计到 2 倍为止，剩下的部分留给经理复核。
- **豁免**——经理把某客户单独拎出来（比如它关门了），让它不拖累你的增长。

## 一句话总结
- **卖货** → 每行一点点。
- **做大** → 超过一个已经考虑了市场和你客户规模的公平门槛。
- **开新客** → 你自己拿下的新生意，头一个季度分成。

一切都是"你做了这件事，所以你赚到了这份钱"——对着公平的门槛、以稳定的 3 个月为基准来衡量，
让你的奖金反映**你自己**的表现，而不是运气或大盘。
